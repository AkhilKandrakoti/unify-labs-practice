Advanced Dashboard Collection
Project Overview

Project 2: Dashboard Collection is an Intermediate-Advanced CSS project showcasing modern web design techniques.
It demonstrates 3D transforms, Glassmorphism, animated borders, responsive grids, interactive accordions, image hover effects, and animated progress bars in a single dashboard layout.

This project is built entirely using HTML and CSS, with no JavaScript dependencies.

Features
1. 3D Flip Card

Fully interactive front-to-back 3D card.

Uses CSS perspective and rotateY for 3D effect.

Includes an animated progress bar.

2. Glassmorphism Card

Modern frosted glass effect using backdrop-filter: blur().

Semi-transparent gradient with hover zoom effect on image icon.

Subtle shadows for depth.

3. Animated Border Card

Card with animated gradient border.

Smooth rotation animation on the border.

Includes live update icon with pulse animation.

4. Responsive Card Grid

Uses CSS Grid repeat(auto-fit, minmax()).

Cards reflow automatically for different screen sizes.

Maintains consistent spacing and alignment.

5. FAQ Accordion

Fully CSS-only accordion with checkbox hack.

Expandable/collapsible items.

Smooth transition for content reveal.

Includes 4 items, all interactive.

Technologies Used

HTML5

CSS3 (Flexbox, Grid, 3D transforms, animations, gradients, Glassmorphism effects)

No JavaScript required

How to Use

Clone or download the repository.

Open index.html in any modern browser (Chrome, Edge, Firefox, Safari).

Hover over the cards to see animations.

Click on FAQ items to expand/collapse content.

Enjoy a fully responsive and interactive dashboard layout.
