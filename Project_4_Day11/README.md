----> Project Overview

This project is a high-fidelity neon-themed launch page built using pure HTML and CSS, focusing on advanced animations, layered visuals, and interactive UI components without relying on JavaScript frameworks. The goal was to demonstrate mastery of modern CSS techniques while maintaining performance and responsiveness.

Apply complex text effects (neon glow & flicker)

Design interactive UI elements (modal popup)

Build responsive, modern layouts using CSS Grid

Manage layering with z-index and positioning
