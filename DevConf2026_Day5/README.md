# DevConf 2026 Website

## Overview
DevConf 2026 is a professional, modern, and fully responsive website built for an international developer conference. This project showcases advanced front-end development techniques, including **CSS animations, hover effects, glassmorphism, and responsive layouts**.

The site provides an engaging experience for users exploring conference details, sessions, speakers, sponsors, and registration.

---

## Features

### Hero Section
- Fade-in and slide-up entrance animation for heading and description.
- Eye-catching call-to-action buttons with smooth hover effects.

### Conference Highlights (Cards)
- Staggered card entrance animations.
- Hover effects with transform (scale) and shadows for an interactive feel.

### Call-to-Action (CTA)
- Glassmorphism effect with backdrop-filter blur.
- Attention-seeker animation (pulse) on the primary button hover.
- Smooth transitions for a polished experience.

### Decorative Enhancements
- Section headings include ::after decorative lines.
- Subtle shadows and backdrop filters applied to content sections.

### Navigation Sidebar
- Fixed sidebar with smooth hover transitions.
- Active page highlighting and responsive collapse on mobile.

### Responsiveness
- Fully responsive design for **mobile (<768px), tablet (768px-1024px), and desktop (>1024px)**.
- Layouts adapt for small screens with stacked cards and collapsible navigation.
- Font sizes and paddings adjusted using CSS variables for responsiveness.

### Additional Pages
- Speakers, Schedule, Sponsors, Registration, and FAQ pages are all styled consistently with the main design.
- Glassmorphism applied where appropriate.
- Cards, tables, and images maintain a professional look across all devices.

### Advanced CSS Techniques
- **Keyframe animations**: fadeSlideUp, cardFadeUp, pulse.
- **Glassmorphism** with `backdrop-filter`.
- **Hover transformations**: translateY + scale.
- **Smooth transitions** for buttons, links, cards, and tables.
- **Decorative pseudo-elements** (`::after`) for headings.
- **Responsive enhancements** for animations and spacing.
